megacrypter
===========

What do you need?


1) Apache (vhost + mod_rewrite + allowoverride)

2) PHP 5.4 (cURL + mcrypt + memcache + composer)

3) MySQL (optional for blacklist).

4) Edit application/config files with 

4.1) miscellaneous.php

4.2) memcache.php

4.3) database.php

4.4) gmail.php


From here you go alone. Good luck!
